# Jenkins CI/CD Deployment Project

## Project Overview

This project demonstrates a complete CI/CD pipeline implementation using Jenkins, GitHub, GitHub Webhooks, Linux, Nginx and AWS EC2.

Two websites were deployed and managed through the CI/CD pipeline:

* FoodHub Restaurant Website
* ShopEase E-Commerce Website

## Technologies Used

* AWS EC2
* Jenkins
* Git & GitHub
* GitHub Webhooks
* Linux (Ubuntu)
* Nginx
* HTML
* CSS

## Project Architecture

GitHub → Jenkins → Target Server → Nginx → Website

## CI/CD Workflow

1. Developer pushes code to GitHub repository.
2. GitHub Webhook triggers Jenkins Pipeline automatically.
3. Jenkins pulls the latest source code.
4. Jenkins deploys the application files to the target server.
5. Nginx serves the website to end users.

## Projects Deployed

### FoodHub Restaurant Website

A responsive restaurant website deployed using Jenkins CI/CD pipeline and hosted on AWS EC2 with Nginx.

### ShopEase E-Commerce Website

An e-commerce website deployed using Jenkins CI/CD pipeline and hosted on AWS EC2 with Nginx.

## Project Deliverables

* Jenkins Server Setup
* Target Server Setup
* SSH Passwordless Authentication
* GitHub Integration
* GitHub Webhook Configuration
* Automated Deployment Pipeline
* Nginx Configuration
* Website Hosting on AWS EC2
